# Rent-A-Car-Management-System
> Desktop application for managing car rentals, clients, and vehicles using C# and SQL Server.

---

## 📌 Description
This project is a desktop application developed in C# (Windows Forms) for managing a car rental business. The system provides functionality for handling clients, cars, prices, and rental operations through an intuitive user interface and a connected SQL Server database.

---

## ⚙️ Features
- 🔐 User authentication (Admin & User login)
- 👤 Client management (Add, Update, Delete, View)
- 🚗 Car management (Add, Update, Delete, View)
- 💰 Price management for each car
- 📅 Rent management system
- Create new rent
- Automatic duration calculation
- Automatic price per day loading
- Validation of dates
- 📊 Rent history visualization
- ✔ Data validation (EGN, email, phone)
- 🚫 Prevents renting already rented cars

---

## 🛠 Technologies Used
- C# (.NET Framework)
- Windows Forms
- Microsoft SQL Server (LocalDB)
- ADO.NET
- SQL

---

🗄 Database
The system uses a relational database with the following main tables:
- Clients
- Cars
- Price
- Rents

Relationships are implemented using primary and foreign keys to ensure data integrity.

---

## ▶️ How to Run
1. Open the project in Visual Studio
2. Restore the database connection (check .mdf file path)
3. Build the solution
4. Run the application

---

## 📌 Notes
- Make sure SQL Server LocalDB is installed
- Update the connection string if needed
- The system automatically filters available cars
